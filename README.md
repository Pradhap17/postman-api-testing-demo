# Postman API Testing Demo

## Project Overview
This project demonstrates REST API testing using Postman on the FakeStore API.

The project includes CRUD API execution, automated validations, response assertions, and API execution reporting.

---

## APIs Covered

- GET Products API
- GET Single Product API
- POST Product API
- PUT Product API
- DELETE Product API

---

## Validations Performed

- Status Code Validation
- Response Time Validation
- Response Body Validation
- JSON Assertions
- CRUD Workflow Validation

---

## Tools Used

- Postman
- REST API
- JSON
- FakeStore API

---

## Project Structure

- Collection → Postman Collection
- Screenshots → API execution screenshots
- Documentation → PDF & DOCX reports
- TestData → Sample request payloads
